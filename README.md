# workout-bros-frontend

**Social fitness app frontend for finding workout/sports partners.**

## Problem
People struggle to stay consistent with workouts when they lack local accountability and community.

## Solution
Workout Bros helps users discover and connect with nearby people for workouts and sports sessions.

## Architecture
**Current stack signals:** Node.js, React

- Social discovery UI
- Profile and activity-matching screens
- Backend-powered interaction flows

## Scale
Designed for community growth with local-network effects.

## Monetization
Freemium social platform with premium discovery/boost features.

## Roadmap
- Group events and challenges
- Reputation/trust signals
- In-app chat and scheduling

## Quick start
```bash
npm install
npm run dev
```
